Modal plugin

If you want to modify the modal with jQuery, this is a fast way of doing so:

// Modify the title of the modal
$(".modal .title").html("New title");

// Change the content
$(".modal .content").html("Some text for the content");

// Modify the main button
$(".modal .primary").html("Accept");
